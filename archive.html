How to add plans to the archive

1. Fill in your daily plan on the main page.

2. Click "Save Today's Plan" to download the HTML file (e.g. 2026-03-17.html).

3. In your GitHub repo, create a folder called plans/ if it does not exist.

4. Upload the saved HTML file into the plans/ folder.

5. Open plans.json in your repo and add the new date to the list.

6. The archive page reads from plans.json and shows all your past plans.
